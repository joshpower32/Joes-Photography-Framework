# joshpower32.github.io
A responsive photography portfolio website for Joe's Photography — featuring a searchable photo gallery powered by the Pexels API, with a full-screen lightbox viewer, and dedicated pages for About, Gallery, Hire, and Contact.

## Selling this site to a client (lead delivery)

The Contact page form is wired to **Web3Forms** so messages email the client.

1. Get a FREE key at [web3forms.com](https://web3forms.com) using the **client's email**.
2. Open `assets/contact/contact.html` and set `WEB3FORMS_KEY` (in the inline script) to that key.
3. Set `FALLBACK_EMAIL` in the same script to the client's email.
4. Test from the live site and confirm the `🔔 NEW MESSAGE` email arrives.

Free tier = 250 submissions/month per key.
